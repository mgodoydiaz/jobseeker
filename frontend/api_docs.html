API Documentation

Welcome to the Job Seeker Assistant API. Here you will find the available endpoints to interact with the service.

POST /api/job_postings/analyze/

Analyzes a job offer from a given URL. This endpoint scrapes the job description, compares it with a user's CV, and returns a match score and a list of missing skills.

Future Endpoints

The following endpoints are planned for future development:

GET /api/job_postings/

Returns a list of all saved job offers for the authenticated user.

POST /api/job_postings/

Saves a new job offer for the authenticated user.

GET /api/job_postings/<id>/

Returns the details of a specific job offer.

PUT/PATCH /api/job_postings/<id>/

Updates a specific job offer.

DELETE /api/job_postings/<id>/

Deletes a specific job offer.
